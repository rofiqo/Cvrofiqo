Tentang

Saya merupakan seorang Mahasiswa Teknik Informatika Universitas Bina Darma. Teknik Informatika merupakan salah satu program studi pada fakultas Ilmu Komputer. Teknik Informatika di Universitas Bina Darma saat ini memiliki tiga konsentrasi (peminatan) yaitu Software, Database, dan Jaringan. Saya sendiri mengambil konsentrasi Software Engineering dan Database. Pada konsentrasi Software saya diajarkan cara membuat beberapa aplikasi, seperti aplikasi Web, Mobile, ataupun aplikasi Desktop. Dan pada mata kuliah Database, saya belajar cara mengelola data manual menjadi data digital, belajar cara membuat database yang baik. Kedua konsentrasi yang saya ambil merupakan konsentrasi yang berhubungan sangat erat, dikarenakan dalam membuat sebuah aplikasi kita memerlukan sebuah Database untuk menampung data yang ada pada aplikasi tersebut.
